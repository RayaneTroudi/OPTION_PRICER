Detailed Description

Represents a Long Call Butterfly Option.

  • This option is constructed from three calls with three different strikes (K1 < K2 < K3).
  • Payoff is: max(S_T - K1, 0) - 2 * max(S_T - K2, 0) + max(S_T - K3, 0).
