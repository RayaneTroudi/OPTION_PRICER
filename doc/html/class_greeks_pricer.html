Detailed Description

Utility class to calculate option Greeks (Delta, Gamma, Vega, etc.) using the Finite Difference Method (FDM) applied to Monte Carlo pricing.

  • It relies on running the MonteCarloPricer multiple times with perturbed input parameters.

Estimates the Delta of the option using the central finite difference formula.

Estimates the Delta of the option using the central finite difference formula (FDM).

Formula: Delta ≈ (V(S + epsilon) - V(S - epsilon)) / (2 * epsilon)

Parameters

epsilonThe small perturbation in the initial price S0 (dS).
